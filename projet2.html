Conception d’une base de données en Python et EasyPhp

Conception d'une base de données sur mesure pour une entreprise fictive, de la modélisation à la création.

Description du Projet

Dans le cadre de ce, nous avons conçu et développé une base de données complète pour informatiser la gestion des approvisionnements et des ventes de sel de la coopérative de l’île de Ré. L’objectif était de structurer les données issues des récoltes des sauniers, du traitement des produits (gros sel et fleur de sel), ainsi que de leur distribution vers les clients professionnels (centrales d’achat). Le projet a été réalisé en plusieurs étapes :

  • Conception du modèle conceptuel de données (MCD) avec Looping MCD à partir de données brutes et de contraintes métier.
  • Génération du schéma relationnel (MLD), puis création automatisée des tables MySQL via script SQL.
  • Insertion manuelle et automatisée (avec Python) de données issues de fichiers CSV et Excel.
  • Rédaction de requêtes SQL complexes pour extraire des indicateurs utiles (ventes par produit, moyenne par client, etc.), incluant des opérations comme GROUP BY, HAVING, vues, et des mises à jour/différences algébriques.
  • Développement d’une interface graphique en Python avec Tkinter

    • Ce projet m’a permis de consolider mes compétences en modélisation de base de données, en requêtage SQL, en automatisation de traitements avec Python, et en création d’interfaces utilisateurs ergonomiques.

    Technologies utilisées

    • MySQL
    • Python
    • Tkinter
    • Looping
    • Excel

    Fonctionnalités clés

    Le système permet de :

    • Création complète de la base de données relationnelle (MCD, MLD, script SQL)
    • Insertion automatique des données à partir de fichiers externes (CSV/Excel)
    • Interface graphique intuitive pour :
      • Visualiser les données
      • Lancer des requêtes prédéfinies
      • Ajouter de nouvelles données
    • Génération de rapports personnalisés pour la coopérative (ventes, stocks, clients)
    • Intégration d’un système de gestion des prix par année et par produit
